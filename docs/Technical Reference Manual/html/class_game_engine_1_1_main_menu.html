GameEngine::MainMenu::MainMenu ( DataPtr  data)

MainMenu Constructor. Changes window object to have dimensions of SCREEN_WIDTH and SCREEN_HEIGHT as defined in DEFINITIONS.h. Sets the window top left position to one third of the monitor width, such that the window is centered (the window width is one third of the monitor width, as defined in DEFINITIONS.h). Creates instance of StateRenderer class for drawing purposes.

Parameters
dataShared pointer to Data layer, used to queue GamePlay state when the user presses the spacebar to begin the game.
