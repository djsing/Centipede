bool  dead 
)

Operator overloading to allow an Entity to be compared to a boolean.

Parameters
entityThe entity being checked.
deadTrue if checking whether the Entity is dead, False if checking whether the Entity is alive.
Returns
Returns dead == entity.IsDead() i.e. If the Entity is being checked to see if it is dead (dead = True): returns True if the Entity object is dead, else False. If the Entity is being checked to see if it is alive (dead = False): returns True if the Entity object is alive, else False.
